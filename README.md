.NET 6 Admin
===============================
_P.S This Admin Tool Project!_

## 1. Technologies and tools used:
* .Net 6
* Mysql 5
* Bootstrap 5
* Jquery 3.6.3
* Visual Studio 2022

## 2. 주요기능: 
* 회원가입, 로그인, ISMS심사 규격, 권한 관리
* Bootstrap 5 Admin Theme, 다국어 처리, 메일링
* LogDiff
